Item Import Help

Import Help

  • For Row Import preview click the 'Submit' button without choosing a File.
  • Item Type selection is ignored by XML Import plugins.
  • Import source owner values are ignored during the import process.

Import Row Help

  • The Item Type is the column in the data which specifies a item type (CD,VHS,VCD,DVD,etc).  The rows will be restricted to only the rows that have the same type as the selected item type.
  • In order for the import to work properly, you should supply the 'value', not the 'display' column from the s_attribute_type_lookup table, otherwise there will be no match.
  • The Default values are only used if a column is not selected from import file.
  • If you do not specify a Category, a empty category will be inserted.
  • 'Initcap' will -except for the first letter in each word- set everything to lowercase. Any uppercase roman numerals up to XX will not be initcapped.

Import XML Help

  • The Status Type value will be used:
    • When no Status Type is provided
    • When the Status Type provided is illegal
    • The Override Status Type is checked
  • Checking the Override Status Type checkbox will cause any import source Status Type's to be ignored.
